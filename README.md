# GribGame - платформер на pygame
2D-платформер, написанный на Python с использованием Pygame. Игрок перемещается по уровням, собирает очки, избегает мобов и добирается до двери.

---

## Геймплей
* 3 уровня с уникальными картами платформ.
* Система жизней (3 жизни) и очков.
* Два вида бонусов: обычные (1 очко) и редкие (3 очка).
* Мобы с патрулированием в заданном диапазоне.
* Фоновая музыка с возможностью включить/выключить.

---

## Управление
* `A/D` - Движение влево / вправо.
* `SPACE` - Прыжок.
* `M` - Включить / выключить музыку.
* `ESC` - Выход в главное меню.
* `Y/N` - Подтверждение / отмена в диалоговых окнах.

---

## Что реализовано
**Игровая механика**
* Физика персонажа: гравитация, прыжок, ограничения по краям экрана.
* Коллизии с платформами: приземление сверху, оттлакивание снизу и по бокам.
* Коллизии с мобами: потеря жизни и сброс позиции игрока.
* Сброс очков при касании бонусов.
* Переход на следующий уровень при касании двери.

**Мобы**
* Патрулируют платформы в заданном диапазоне координат.
* Подтверждение вызода в главное меню.

**HUD**
* Счетчик очков в левом верхнем углу.
* Иконки жизней в правом верхнем углу.
